How to run the code:
1. cmake .
2. make
3. ./bin/MotionPlanner scene3d.txt 120
4. after opening the application, right click and press "Run motion planner". The path will be built and shown in window after a while. The time used will be shown in the terminal.
5. Then, right click, press "Get highest reward path". The robot will run through the path  in animation.
